# AI Data Engineer Agent

An AI-powered data engineering assistant that helps users upload, profile, clean, analyze, and query datasets.

## Features

- Upload CSV and Excel files
- Dataset preview
- Missing value detection
- Duplicate row detection
- Data type analysis
- Basic statistics
- AI-based cleaning suggestions
- SQL query generation
- Interactive dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- SQLite / PostgreSQL
- LLM API / Ollama

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/main.py
```
